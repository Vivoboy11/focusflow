FocusFlow: The Persistent Offline Timetable
A Python-based productivity tool developed for student workflow.

🚀 Features
Auto-Rollover: Unfinished tasks from previous days automatically move to today's schedule.

Priority Levels: Use !!!, !!, or ! to rank your study tasks.

Focus Timer: Integrated Pomodoro timer with custom duration support.

Offline First: All data is stored locally in tasks.json.

📖 Usage
add "Study Python" --p 3 - Adds a high-priority task.

focus 15 - Starts a 15-minute study timer.

del 1 - Deletes the first task on the list.
